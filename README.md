# Sanaa Mouzahir

PhD candidate at MIT — [MSEAS Lab](https://mseas.mit.edu), Mechanical Engineering +
Center for Computational Science and Engineering. I build machine-learned closure
models that make coarse simulations of turbulence behave like fine, high order computationally expensive ones.

## Research

- **[qg-closure](https://github.com/sanaamouzahir/qg-closure)** — ML temporal &
  spatial closures for 2D quasi-geostrophic turbulence: local-truncation-error
  theory, Wiener-conditioned stencils, von Neumann–certified stability.
  Manuscript in [`paper/`](https://github.com/sanaamouzahir/qg-closure/tree/main/paper).
- **[CLoVER-gp](https://github.com/sanaamouzahir/CLoVER-gp)** — probabilistic
  closures for coarse-grid Navier–Stokes (CNN + deep-kernel SVGP, calibrated
  uncertainty), deployed online inside a MATLAB finite-volume solver.
- **[heart-valve-shape-optimization](https://github.com/sanaamouzahir/heart-valve-shape-optimization)** —
  automated pediatric heart-valve design: NURBS geometry → CAD → meshing →
  LS-DYNA → shape optimization.
- **[bridge-surrogate-fem-optimization](https://github.com/sanaamouzahir/bridge-surrogate-fem-optimization)** —
  surrogate finite-element bridge model calibrated with a genetic algorithm.
